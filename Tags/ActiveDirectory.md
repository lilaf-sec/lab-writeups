# Active Directory

---

## 📌 Definition
> **Active Directory (AD)** is a directory service developed by Microsoft for Windows domain networks. It manages authentication, authorization, users, computers, groups, and policies within a centralized domain environment.

---

## 🛠️ Impact
- Credential compromise and lateral movement within the domain
- Privilege escalation to Domain Admin
- Abuse of Kerberos authentication (e.g., ticket attacks)
- Dumping hashes and performing Pass-the-Hash / Pass-the-Ticket attacks
- Full domain compromise if misconfigurations are exploited
